# 🛒 Caixa Registradora em Java

Projeto simples desenvolvido em Java com o objetivo de praticar os conceitos fundamentais da linguagem.

## 📚 Conceitos utilizados

* Variáveis
* Tipos primitivos
* Operadores matemáticos
* Scanner
* Entrada e saída de dados
* Estruturas condicionais (if/else)

## 🚀 Funcionalidades

O sistema permite:

* Informar o nome do produto
* Informar o valor do produto
* Informar a quantidade comprada
* Informar o valor pago pelo cliente
* Calcular o valor total da compra
* Verificar se o valor recebido é suficiente
* Calcular e exibir o troco

## 💻 Exemplo de execução

Digite o Nome do produto:
Arroz

Digite o valor do produto:
25

Digite a quantidade de produto:
2

Digite o valor recebido:
60

====== Compra Concluída ======

Produto: Arroz
Quantidade: 2
Valor Recebido: 60
Valor Final: 50
Troco: 10
